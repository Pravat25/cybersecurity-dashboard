## Overview

Cybersecurity Dashboard is a Flask-based web application designed to monitor and display network activity logs in real time.

The dashboard reads packet logs generated by a network packet sniffer and presents them in a clean web interface. This simulates a basic **Security Operations Center (SOC) monitoring system**.

---

## Features

- Real-time log monitoring
- Displays network packet activity in browser
- Auto-refresh dashboard (every 5 seconds)
- Simple and clean UI
- Integration with packet sniffer logs

---

## Technologies Used

- Python
- Flask
- HTML (Jinja2 templates)

## Installation

Clone the repository:


git clone https://github.com/Pravat25/cybersecurity-dashboard.git


Navigate into the project folder:


cd cybersecurity-dashboard


Create virtual environment:


python3 -m venv venv


Activate it:


source venv/bin/activate


Install dependencies:


pip install flask


---

## How to Run

Run the application:


python app.py


Open your browser:


http://127.0.0.1:5000

## Integration with Packet Sniffer
